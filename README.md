### 🛒 Multimodal Recommendation Engine

> Vector search engine catalog scalability

`AWS SageMaker` `CLIP Embeddings` `OpenSearch HNSW` `WandB`

Built an AWS-native pipeline utilizing CLIP multimodal embeddings (image + text) with HNSW graphs on OpenSearch. Successfully reduced catalog search complexity from $O(N)$ to $O(\log N)$ for production-scale pipelines.

### 🩺 AI-Driven Mobile Thermography

> Edge AI & Model Optimization for Clinical Environments

`Edge AI` `Model Quantization` `XNNPACK` `Mobile Deployment`

Developed an optimized mobile solution for thermographic imaging in healthcare. Implemented severe model quantization (INT8/FP16) leveraging the **XNNPACK** backend to achieve ultra-low latency and low memory footprint during real-time edge inference in clinical settings.
